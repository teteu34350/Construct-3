# Construct-3

Repositório destinado ao armazenamento e organização de um jogo desenvolvido na plataforma Construct 3.

Este projeto possui fins acadêmicos e de aprendizado, com foco na prática de lógica de programação aplicada a jogos, estruturação de fases, mecânicas interativas e organização de projetos de game design.

O jogo principal, **SuperNeyko**, é um jogo de plataforma inspirado no universo do futebol, trazendo um personagem com habilidades especiais em um ambiente dinâmico. O projeto tem como objetivo explorar conceitos fundamentais do desenvolvimento de jogos, como movimentação, colisões, progressão de fases e interação com o jogador.

O desenvolvimento foi realizado com foco educacional, visando aprimorar habilidades em criação de jogos 2D e design de níveis utilizando a plataforma Construct 3.
